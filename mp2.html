MP2 Write-Up: P5 Gallery

1. Overview & Usage:
My project is a collection of different P5 programs that includes the following webpage mediums and styles: static, randomized static, infinite-loop, interactive, audio, and audio-visual interaction. Throughout the process of advancing stages, each one had shared code structure and syntax, allowing for the programs to build off each other. The last stage, audio-visual interaction, is a project that cohesively covers the aforementioned stages’ ideas.
2. Development Process
Static:
• I used the P5 web editor to run my code and debug issues before adding any into my site code.
• I started a file for the static project. Each had a created canvas and a background color.
• I set relevant global variables that appear throughout the code.
• I created a for-loop that would repeat the draw function.
• This was enabled by a call to another function.
• The called function created the content in terms of the color, framing, number, and shape.
Randomized Static:
• Following, for the random static project, I set a random seed value, which acts as a random number generator.
• I created a draw function that iterates, assigning randomized value to the shape variables.
• This determined the coordinates and color values of the shapes.
Infinite-Loop:
• Next, for the infinite-loop project, I set relevant global variables.
• The setup function iterates, assigning randomized value to the shape variables.
• Within an array, we push and save the randomized shapes’ values.
• Using the canvas coordinates, we assign the center of rotation.
• With the randomized variables and center coordinates, I created a function that iterates, determining the diameter, position, and speed.
Interactive:
• After, for the interactive project, I set an array as a global variable and created a canvas.
• I set a variable that would represent the completion time per second or FPS.
• Through an iterative process, the array would be updated with frames, which had a set timer based on the array length.
• For the next function, I used the canvas coordinates and frameCount function to push the frames onto the canvas as visual graphics.
• Additionally, there was a class used to represent the shape’s framing, color, lifespan, and position.
• Try dragging your mouse on the canvas below!
Audio:
• For the audio project, I created a HTML file that constructed a table with buttons.
• For each button, on click of the user’s mouse, the JavaScript function is called and executed.
• Following, I created a CSS file that would contain the code that determined the font properties as well as the soundboard interface design.
• Afterwards, I created a JS file that would contain the code that determined the behavior of each button, which would execute a function that would play a sound through assigning local audio mp3 files as variables.
Audio-Visual Interaction:
• Finally, for the audio-visual interaction project, I took parts of my code from the audio and infinite loop project as a foundation for the program.
