MP2 Write-Up: P5 Gallery

1. Overview & Usage:
My project is a collection of different P5 programs that includes the following webpage mediums and styles: static, randomized static, infinite-loop, interactive, audio, and audio-visual interaction. Throughout the process of advancing stages, each one had shared code structure and syntax, allowing for the programs to build off each other. The last stage, audio-visual interaction, is a project that cohesively covers the aforementioned stages’ ideas.
2. Development Process
Static:
• I used the P5 web editor to run my code and debug issues before adding any into my site code.
• I started a file for the static project. Each had a created canvas and a background color.
• I set relevant global variables that appear throughout the code.
• I created a for-loop that would repeat the draw function.
• This was enabled by a call to another function.
• The called function created the content in terms of the color, framing, number, and shape.
• Following, for the random static project, I set a random seed value, which acts as a random number generator.
• I created a draw function that iterates, assigning randomized value to the shape variables.
• This determined the coordinates and color values of the shapes.
